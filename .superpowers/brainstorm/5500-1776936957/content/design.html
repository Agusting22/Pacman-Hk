🎮 Core Loop

Flujo de una vida
🍌 Banana aparece → 💰 Recogés pesos → 🦹 Chorro te persigue → 👮 Agarrás una cana → 🦹 Chorro huye 5s → 🚇 Llegás al subte = WIN
⚠️ Si el chorro te toca → perdés una vida (empezás con 3). Sin vidas → Game Over.

⚙️ Stack Técnico

Arquitectura del proyecto
📁 juego-hackathon/
  📄 index.html        ← Entry point, carga Phaser CDN
  📄 game.js          ← Todo el juego (~30-40KB)
  🖼️ hackathon.png     ← El chorro (asset local)
Phaser 3 → CDN (no cuenta para los 50KB)
game.js contiene: Scenes (Menu, Nivel1, Nivel2, Nivel3, GameOver, Win), lógica de AI chorros (pathfinding simple BFS), generación de mapas, HUD
